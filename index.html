Hello People of the Internet!

This is the home page for my Geoffamania! games.

Geoff's Who! - This is a Game Board to help aid with creating/hosting Guess Who, complete with a functional timer and two image upload locations for branding.

Tic Tac Oh No! - This is a Game Board for Tic Tac Oh No - my take on Tic Tac Toe using bluffing mechanics and wrestling knowledge. I'll be using the framework of this game for two upcoming spinoff games, Work or Shoot? and WrestlePath.

Geo(ff)pardy! - This is the Game Board for Geo(ff)pardy, a survival poker take on the classic trivia game. This comes complete with a full ruleset, trivia fed by a linked Google Sheet, automated scoring logic, a timer, and Bonus/Curse elements.

Work or Shoot - This is the Game Board for Work or Shoot, a Connect Four-like game where players need to create a four-wrestler connection based on played wrestlers' wins. This is the online/social version - solo/app version to come soon!
